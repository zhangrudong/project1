电商项目：
	水果网站
本地利用gulp搭建以server文件夹服务器；
date文件夹：模拟的json数据；
libs文件夹：引用的外部文件；
js文件夹：main.js是js合并文件，main.min.js为压缩后的文件；
首页文件：index.html;
用到sass写部分页面css；

打开首页html文件，即可浏览页面，
功能：
1.注册，用户名必须为手机号，密码为6-18位数字或字母；
2.登录，用户名密码不符会提示错误信息，登录成功跳转到首页，显示欢迎***登录；
3.搜索下拉框，鼠标键盘可操作；
4.导航三级菜单，划过显示，点击跳转商品列表；
5.商品列表，模拟数据加载页面，划过显示按钮，点击商品名称跳转详情页，点击按钮加入购物车；
6.购物车，未登录状态下自动跳转登录页面，登录成功才能进入购物车页面；
7.商品详情页，鼠标划过商品图片自动放大；
